Contents:
- creerVehicule4.m            : create vehicle parameters (4-wheel front-drive)
- creerSim.m                  : simulation setup (speed, steering profile, drive profile)
- creerPneumatique_simple.m   : simple tire parameters
- modele_cinematique_4roues.m : kinematic model (for comparison)
- model_dynamique_4roues.m    : dynamic 3-DOF model (front-wheel drive)
- Main_4roues.m               : example script to run the models and plot comparisons

Instructions:
1) Place all .m files in the same folder.
2) Open MATLAB and set the current folder to that folder.
3) Run Main_4roues.m.
4) Adjust parameters in Main_4roues.m as needed (speed, steering, torque).
